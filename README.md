#Sportify Planner

Descripció

Sportify Planner és una aplicació web dissenyada per ajudar als usuaris a gestionar els seus entrenaments, metes esportives i seguiment de l'activitat física de manera senzilla i eficaç. Permet crear plans d'entrenament personalitzats, buscar exercicis a través d'una API d'exercicis, gestionar les teves metes i veure'n l'evolució. També inclou una secció de blog amb articles sobre consells esportius, preparació física i altres temes relacionats.

#Característiques principals

Plans d'entrenament personalitzats: Crea i segueix el teu propi pla d'entrenament.
Cerca d'exercicis: Utilitza la API d'ExerciseDB per cercar exercicis segons les teves metes.
Blog: Llegeix articles relacionats amb l'entrenament, la salut i la nutrició.
Gestió de metes: Controla les teves metes esportives i personalitza els objectius.
Interfície responsiva: Adaptació a dispositius mòbils per garantir una bona experiència d'usuari en diferents pantalles.
Efets de hover i interacció: Millora l'experiència d'usuari amb transicions suaus i efectes dinàmics.
Tecnologies utilitzades

Vue.js: Per a la creació de la interfície d'usuari reactiva.
Pinia: Gestió de l'estat global de l'aplicació, com la selecció de les metes i els exercicis obtinguts.
Tailwind CSS: Per al disseny responsiu i flexible, facilitant la creació d'interfícies d'usuari modernes.
Firebase: Per gestionar el backend, autenticació i emmagatzematge de dades.
Rapid API (ExerciseDB): API utilitzada per cercar exercicis en funció de les metes esportives dels usuaris.

#Per començar a utilitzar l'aplicació en el teu entorn local:

Clona el repositori:
git clone https://github.com/usuari/Sportify-Planner.git
Instal·la les dependències:
npm install
Inicia l'aplicació:
npm run dev
Obre el teu navegador i ves a http://localhost:... per veure l'aplicació en funcionament.
